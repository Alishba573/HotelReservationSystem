# Hotel Reservation System - Software Construction CCP

## Project Overview
This is a hotel room reservation system implemented in Java as part of the Complex Computing Problem (CCP) for Software Construction course.

## Domain Classes Implemented (Day 1)
- `Hotel` - Represents a hotel with rooms
- `Room` - Represents individual hotel rooms
- `Customer` - Stores customer information
- `Booking` - Manages room bookings

## Technology Stack
- Java 17
- Object-Oriented Design Principles
- Defensive Programming
- JUnit 5 (to be added)
